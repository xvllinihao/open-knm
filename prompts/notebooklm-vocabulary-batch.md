# 分批生成荷兰语词汇表

由于输出限制，我们分批生成。每次生成 **50-100 个单词**。

## 当前批次：第 X 批（共 5-10 批）

**任务**: 生成 50-100 个荷兰语单词，JSON格式。

**格式**:
```json
[{"id":"id","dutch":"荷兰语","category":"daily|work|housing|health|admin","level":"A2","translations":{"en":"英文","zh":"中文"},"notes":{"en":"注释","zh":"注释"}}]
```

**分类**:
- daily: 日常(购物/交通/天气/食物/社交)
- work: 工作(求职/办公室/会议/合同)
- housing: 住房(租房/买房/维修/家具)
- health: 医疗(看病/保险/症状/药物)
- admin: 行政(政府/证件/税务/注册)

**要求**:
1. 实用优先，荷兰特色优先
2. 假朋友标注"FALSE FRIEND"
3. notes简洁但包含关键信息
4. 避免已生成批次中的重复
